Libraries
DataForMe exists thanks to these open-source libraries:
  • webpack - an open-source module bundler for JavaScript files
  • frappe-charts - open-source svg graphing library
  • zip.js - open-source library for manipulating zip files in the browser
  • lottie - airbnb animation library used just for nav
  • feather-icons - simply beautiful open-source icons
  • fontawesome - for a couple of awesome icons
Inspirations
Some places where I looked for inspiration (not affiliated with any of these websites):
  • last.fm - keeps track of your music listening habits. The clock graph that you see in the reports is inspired by this website.
  • stripe.com - just an overall beautiful, simple and fast website. Found myself looking through their markup and styles multiple times.
